celery Package¶

mediagoblin.init.celery.setup_celery_from_config(app_config, global_config, settings_module='mediagoblin.init.celery.dummy_settings_module', force_celery_always_eager=False, set_environ=True)[source]¶

Take a mediagoblin app config and try to set up a celery settings module from this.

Args: - app_config: the application config section - global_config: the entire ConfigObj loaded config, all sections - settings_module: the module to populate, as a string - force_celery_always_eager: whether or not to force celery into

always eager mode; good for development and small installs
  • set_environ: if set, this will CELERY_CONFIG_MODULE to the settings_module

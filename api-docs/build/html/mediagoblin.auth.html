mediagoblin.auth.lib.fake_login_attempt()[source]¶

Pretend we’re trying to login.

Nothing actually happens here, we’re just trying to take up some time, approximately the same amount of time as bcrypt_check_password, so as to avoid figuring out what users are on the system by intentionally faking logins a bunch of times.
